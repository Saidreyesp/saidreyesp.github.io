Sobre este Proyecto

Este sitio web fue creado como parte de una actividad de un curso de desarrollo web. El objetivo es aplicar conocimientos de HTML5 y CSS para crear una página estructurada y llamativa.

La página incluye elementos semánticos, multimedia, listas, tablas y formularios.

Tecnologías Utilizadas

Para este proyecto se utilizaron las siguientes tecnologías: HTML5 para la estructura, CSS3 para los estilos, y GitHub Pages para el hosting.
